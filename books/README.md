# Книги с данными о продуктах

Положите сюда MD-файлы с таблицами продуктов питания.

## Формат

Скрипт `tools/parse-books.js` ищет Markdown-таблицы с разделителями `|`.

Пример таблицы:

| Продукт | Калории | Белки | Жиры | Углеводы |
|---------|---------|-------|------|----------|
| Овсянка | 68      | 2.4   | 1.4  | 12       |
| Гречка  | 110     | 4.5   | 2.3  | 25       |

Скрипт автоматически определяет колонки по заголовкам.

Поддерживаемые колонки:
- Название продукта
- Калории (ккал)
- Белки (г)
- Жиры (г)
- Углеводы (г)
- Быстрые углеводы (г)
- Аминокислоты: лейцин, изолейцин, валин, лизин, метионин, фенилаланин, треонин, триптофан, гистидин
- Омега-3, Омега-6

## Запуск парсинга

```bash
node tools/parse-books.js
```

Результат будет записан в `data/products.json`.
